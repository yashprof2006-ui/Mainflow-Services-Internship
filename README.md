## 📊 Completed Tasks

### Task 1 – Customer Churn Prediction

#### Objective
Build a predictive analytics solution to identify customers who are likely to leave a telecom service provider.

#### Work Performed
- Data Understanding
- Data Cleaning
- Handling Missing Values
- Exploratory Data Analysis (EDA)
- Churn Analysis
- Correlation Analysis
- Data Visualization

#### Key Insights
- Customers with Month-to-Month contracts show higher churn rates.
- Customers with shorter tenure are more likely to churn.
- Monthly charges significantly influence customer retention.
- Correlation analysis helps identify potential predictors of churn.
